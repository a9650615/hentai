紳士抓圖器 exhentai網站抓圖器
===================
此程式是基於 NW ( NodeWebkit ) 開發


----------

環境準備
----
方法一.npm 安裝法 ( node_module 包含奇怪的東西 不方便使用)

方法二.直接去下載nw
http://nwjs.io/

把解壓縮檔的內容( 不包含資料夾 )丟至程式資料夾中，執行 nw 執行檔即可


----------


###開發工具

 1. NW 程式主框架
 2. HTML,Jquery,CSS 程式介面
 3. mkdirp (node module) 建立資料夾
 4. request (node module) 發送請求
